# Home Loans Project Analytics By Power BI

## Overview

This repository contains a Power BI dashboard for a detailed analytical view into home loan data. The goal is to provide actionable insights into loan distributions, approval rates, risk assessments, and customer segmentation that can aid financial institutions in making informed decisions.

## Features

- **Interactive Dashboards**: A comprehensive set of interactive dashboards providing insights at a glance.
- **Approval and Default Analysis**: Visualizations focused on approval rates by demographics and default rates by loan characteristics.
- **Customer Profiles**: Analytical breakdowns that categorize customers based on loan amount, income, and other relevant factors.
- **Risk Metrics**: Metrics and visualizations that identify and analyze loan risk factors.

## Visualizations Included

- Distribution of Loan Amounts
- Approval Rates by Credit Score and Demographics
- Default Rates by Purpose of Loan
- Geographic Distribution of Loans
- Correlation between Applicant Income and Loan Amount

## Data

The project uses a dataset encompassing various loan application details, including applicant income, loan amounts, credit history, property details, and loan approval status. Data integrity is maintained through careful cleaning and transformation processes.

## Tools

- **Microsoft Power BI**: Utilized for developing interactive visualizations and comprehensive dashboards.
- **DAX (Data Analysis Expressions)**: Employed for crafting calculated columns, custom measures, and tables within Power BI.
- **Power Query**: Applied for data merging, cleansing, and transformation tasks.

## Usage

To use this dashboard:
1. Clone the repository to your local machine.
2. Open the `.pbix` file using Microsoft Power BI Desktop.
3. Explore the dashboard by selecting different filters and slicers to drill down into specific data points.
